# Getting Started
## Flutter Setup
https://docs.flutter.dev/get-started/install

## Setup
run make setup at the project root.
```
make setup
```

## Commands
### dependencies
run make dependencies is call flutter pub get.

```
make dependencies
```
### build-runner
run make build-runner is call flutter packages pub run build_runner build.
```
build-runner
```
